Tecnologias Utilizadas:
Microsoft ASP.NET MVC 4 / C#
Microsoft EntityFramework 5
jQuery 1.11
jQueryUI 1.9
bootstrap 3.0
Noty 2.3.4
BlockUI 2.70

Requisitos:
1- Visual Studio 2013 
2- SQL LocalDB Express (http://www.microsoft.com/en-us/download/confirmation.aspx?id=29062&6B49FDFB-8E5B-4B07-BC31-15695C5A2143=1 / 32mb)

Executando pelo Visual Studio:
1- Abra o arquivo FaleMais.sln pelo Visual Studio.
2- Defina a aplicação FaleMais.App como aplicação inicial
3- Execute

Fluxo de Uso:
1- /Home/ 
	Tela inicial onde o usuario realiza os calculos
	
2- /Admin
	Tela de administração para gerenciar os preços e planos
	Usuario: admin
	Senha: 123

Executante Teste de Unidade pelo Visual Studio:
1- Abra o arquivo App.config
2- Altere o caminho fisico do arquivo MDF de acordo com a localização em seu computador.
3- Menu "Test" > "Run" > "All Tests"
4- Acompanhe o resultado do teste na janela "Test Explorer"
